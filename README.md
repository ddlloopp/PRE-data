# PRE-data
the database for article "Unifying the Mechanism of the Ionospheric Pre-Reversal Enhancement: The Role of Local Time Variation of Effective Zonal Winds"
defualt.nc, nohal.nc, zonalwind-200.nc, case3.nc, case4.nc corresponds to the experiment data from Run 0 to Run 4 in this article. These files contains the data of zonal winds, zonal electric fields and vertical electric fields.
